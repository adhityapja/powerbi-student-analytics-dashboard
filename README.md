# Student Performance Analysis Dashboard (Power BI)

## Project Overview

This project is a Power BI dashboard that analyzes student performance based on marks, attendance, department, subject, and semester. The dashboard helps in identifying student performance trends, attendance impact on marks, and subject-wise performance.

## Dashboard Pages

The report contains 4 pages:

1. Overview Dashboard
2. Subject Analysis Dashboard
3. Student Performance Analysis
4. Attendance Analysis

## Key Insights

* Students with higher attendance tend to score higher marks.
* Some subjects have higher failure rates compared to others.
* Department-wise performance comparison is included.
* Attendance categories are classified as High, Medium, and Low.

## Tools Used

* Power BI
* Excel Dataset
* DAX (Data Analysis Expressions)

## Features

* KPI Cards
* Bar Charts
* Pie Charts
* Scatter Plots
* Filters (Department, Gender, Subject, Semester)
* Interactive Dashboard
